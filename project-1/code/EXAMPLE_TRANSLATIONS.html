How to Use This System

  1. Select a language from the dropdown in the top-right corner
  2. The page text will automatically update to the selected language
  3. Your language preference is saved automatically
  4. Visit any page and it will remember your language choice

For Developers: Implementation

To add translations to your HTML elements, use the data-t attribute:

<span data-t="dashboard">Dashboard</span> <button data-t="create">Create</button> <a href="#" data-t="profile">Profile</a>

To programmatically change language:

setLanguage('en'); // English setLanguage('es'); // Spanish setLanguage('fr'); // French setLanguage('pt'); // Portuguese
